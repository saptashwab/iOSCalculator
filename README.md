# iOSCalculator

<h2>Calculator App</h2>

This is an XCode project of a calculator app. It has the following functions:
<ul>
<li>Change sign (±)</li>
<li>Square root (√) </li>
<li>Addition (+)</li>
<li>Subtraction (-)</li>
<li>Multiplication (×)</li>
<li>Division (÷)</li>
<li>Cosine (cos)</li>
<li>Sine (sin)</li>
<li>Logarithm (log)</li>
<li>Natural logarithm (ln)</li>
</ul>
